# todo_app_test
[todo_app](https://github.com/ryusuke-homma-unifa/ryusuke-homma-unifa.github.io/tree/master/todo_app)のテストです

# 実行手順
1. [cypress](https://www.cypress.io/)をインストールしてください
1. 本リポジトリをクローンし、`npm run cy:run` コマンドを実行してください
